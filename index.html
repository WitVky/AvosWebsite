What is AVoS Dialer

AVoS is software suite developed by Irisind to automate and handle large number of calls efficiently. It primarily aims at providing efficient communication and management services for businesses and call centers. AVoS Dialer is a web app that is designed to interact with the Asterisk Open-Source PBX Phone system to act as a complete outbound call center software. It automatically dials the numbers and assigns available agents to the calls. It can also send dropped calls to the voicemail and schedule a callback.

AVoS administrators will have a simple yet powerful web-based interface through which they can monitor real-time activity, review summary reports, keep track of agents, manage campaigns and do much more.

The agent interface is an interactive web page that works through a web browser to give real-time information and functionality with nothing more than an internet browser on the client computer.
